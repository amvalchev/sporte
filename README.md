# sporte

Sporte is a simple web application that displays a sports event calendar.

## Table of Contents
- [Installation](#installation)


#Installation

## Prerequisites
You have to have Go installed, you can download it from the official site for your system - https://go.dev/dl/

1. git clone https://github.com/amvalchev/sporte
2. cd sporte
3. go mod download
4. go mod tidy
5. go run ./cmd/web
